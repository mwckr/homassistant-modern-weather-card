# Security Policy

## Supported Versions

Currently, only the latest release of the Zuhause Weather Card receives security updates.

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please do not disclose it publicly. 

Instead, please report it privately via a [GitHub Security Advisory](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing/privately-reporting-a-security-vulnerability) in the repository or contact the repository owner directly.

We will review the issue and respond as swiftly as possible with a timeline for a fix.

Any other issue can be publicly reported here: [New issue](https://github.com/mwckr/homassistant-modern-weather-card/issues/new)
